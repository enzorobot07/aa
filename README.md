## Quick Install (Recommended) - Using Pre-built Binaries
Download and install the latest pre-built binary for your architecture:

**For AMD64 (x86_64):**
```shell
curl -L https://github.com/enzorobot07/BillionMail/releases/latest/download/billionmail-amd64 -o /usr/local/bin/billionmail
chmod +x /usr/local/bin/billionmail
billionmail --help
```

**For ARM64:**
```shell
curl -L https://github.com/enzorobot07/BillionMail/releases/latest/download/billionmail-arm64 -o /usr/local/bin/billionmail
chmod +x /usr/local/bin/billionmail
billionmail --help
```
